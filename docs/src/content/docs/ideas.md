AmbientNav is an open-source project shaped by its community. This page collects ideas for new features, modules, and hardware integrations that have not yet been implemented.

Have a suggestion? [Submit it on GitHub](#submit-your-idea) — the community votes with 👍 reactions and the most popular ideas get prioritised for upcoming releases.

## Submit Your Idea

1. Open a new issue using the **[💡 Idea template](https://github.com/vergissberlin/ambientnav/issues/new?template=idea.yml)** on GitHub.
2. Fill in the category, description, and use case.
3. Submit — anyone can vote with a 👍 reaction.

Ideas with the most reactions are reviewed for the next release cycle.
