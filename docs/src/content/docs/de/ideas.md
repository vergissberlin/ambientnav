AmbientNav ist ein Open-Source-Projekt, das von seiner Community geprägt wird. Diese Seite sammelt Ideen für neue Features, Module und Hardware-Integrationen, die noch nicht umgesetzt wurden.

Du hast einen Vorschlag? [Reiche ihn auf GitHub ein](#idee-einreichen) — die Community stimmt mit 👍-Reaktionen ab, und die beliebtesten Ideen werden für kommende Releases priorisiert.

## Features & Sicherheit

| Status | Idee |
|--------|------|
| ☐ | Warnung bei überhöhter Geschwindigkeit — visueller Hinweis beim Überschreiten eines konfigurierten Limits |
| ☐ | Warnung vor Blitzern |
| ☐ | Hinweis bei Unfällen auf der Route |
| ☐ | Warnung vor Glatteis |
| ☐ | Konfiguration von verschiedenen Ambient-Szenen |
| ☐ | Module einzeln deaktivierbar (z. B. Einparkhilfe deaktivieren, wenn man einen Anhänger hat) |
| ☐ | Display in der Heckscheibe: *„Ihr Abstand ist zu gering"* |
| ☐ | Siri-Steuerung |
| ☐ | Fahrtenbuch |
| ☐ | Rückfahrkamera-Integration |

## Sensoren & Hardware-Module

| Status | Idee |
|--------|------|
| ☐ | Breitere LED-Streifen für Seiten- und Unterbodenbeleuchtung |
| ☐ | Mikrofon — Umgebungsgeräusche als Eingabe für Lichteffekte nutzen |
| ☐ | Lichtsensor zur automatischen Abdimmung |
| ☐ | *„Hey KIT"* — Knight-Rider-ähnliche KI-Sprachassistent-Integration |
| ☐ | Beschleunigungssensor — Wassersimulation bei starkem Bremsen, Feuereffekt beim Anfahren |

## Idee einreichen

1. Öffne ein neues Issue mit der **[💡 Ideen-Vorlage](https://github.com/vergissberlin/ambientnav/issues/new?template=idea.yml)** auf GitHub.
2. Fülle Kategorie, Beschreibung und Anwendungsfall aus.
3. Absenden — andere können mit 👍 abstimmen.

Ideen mit den meisten Reaktionen werden für den nächsten Release-Zyklus berücksichtigt.
