The AmbientNav action button — use for any tap/click action; variant follows the light language.

```jsx
<Button variant="primary" iconRight={<ArrowRight/>}>Start guide</Button>
```

Variants: `primary` (cyan, default action), `alert` (magenta, stop/warning), `gradient` (brand hero CTA), `secondary` (filled neutral), `ghost` (hairline outline). Sizes `sm | md | lg`. Press scales to 0.97. Don't mix cyan and magenta buttons for mere variety — the hue signals intent.
